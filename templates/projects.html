SubCanvas

SubCanvas Website
GitHub Repository

Subcanvas is a website where an image is initially obscured and divided into small sections. Throughout the month, users a from specific Twitch chat can reveal sections of the image by subscribing or donating, with a new image presented each month. I developed the backend for this project, handling subscription and donation detection, and exposing data via APIs. The project was created in collaboration with Enkk (a streamer who uses this project) and lorenzozane.

MyIpTracker

GitHub Repository

MyIpTracker is a Python script that notifies the user when their public IP address changes. It does this by checking the public IP address and comparing it to the previous one, and if it detects a change, it sends a message to the user via Telegram bot.
